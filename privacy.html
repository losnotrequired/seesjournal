Privacy

This is a small, static website. It does not set advertising or tracking cookies and does not build profiles of visitors.

Forms

If you submit the contact or newsletter form, the information you enter — such as your name and email address — is sent to the site owner through the hosting provider’s form service so we can reply or send updates. We use it only for that purpose and do not sell it.

Fonts

Typefaces load from Google Fonts, which may receive your browser’s request for the font files. If you prefer, the fonts can be self-hosted to avoid this.

Hosting

The site is hosted on Netlify, which keeps standard server logs (for example, IP address and request time) for security and operations.
